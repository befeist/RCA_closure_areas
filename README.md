# RCA_closure_areas
The purpose of this repo is to house all things related to mapping the changing footprint of the groundfish trawl Rockfish Conservation Areas (RCAs) on the West Coast of the United States. We have created geospatial data layers of these dynamic marine protected areas (MPAs), though, they don't qualify as such based on IUCN definitions. The existing data layers are in ArcGIS geodatabase format, created from line features defined in the federal register. Moving forward, this repo will house code written to automate the data layer and attribute creation from the written description.
